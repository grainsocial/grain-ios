# Grain for iOS

A photography community built on the [AT Protocol](https://atproto.com). Share your work, follow photographers, and explore galleries with rich camera metadata.

## Features

- Feed
- Galleries
- Stories
- Camera & EXIF metadata
- Likes, comments & shares
- Notifications
- Search
- Profiles
